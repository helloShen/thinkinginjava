#### Holder.java



```java


package com.ciaoshen.thinkinjava.chapter15;

public class Holder<T> {
    private T obj;
    public Holder(T t){obj=t;}
    public void set(T obj) { this.obj = obj; }
    public T get() { return obj; }
    public String toString(){return obj.toString();}
}


```
