  (2) Demonstrate array covariance using Numbers and Integers.
---



```java


public class Exercise26{
    public static void main(String[] args){
        Number[] numArray=new Integer[10];
        for(int i=0;i<10;i++){
            numArray[i]=new Integer(i);
        }
        for(int i=0;i<10;i++){
            System.out.println(numArray[i]);
        }
    }
}


```
